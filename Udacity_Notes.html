Important Concepts to Remember


Lesson 1: The Basics of the Web and HTML

World Wide Web

The web is simply a vast number of computers that communicate with each other using HTTP or Hypertext Transfer Protocol.

HTML

HTML stand for Hypertext Markup Language. A majority of the content on the World Wide Web is made up of HTML documents. These documents can contain text, images, videos, etc. The text in a HTML document defines what the user sees and the markup defines how the content appears and is arranged.

Tags and Elements

HTML documents consist of elements. HTML tags tell the browser the type of element. Elements are either Inline or Block. From the W3Schools reference: an inline element does not start on a new line and only takes up as much width as necessary; while a block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can)

HTML-CSS-DOM

HTML is a language (syntax & rules). The basic building blocks of an HTML document are tags. Tags are interpreted by the browser (using DOM) to create elements in a tree. An element is an opening tag + content + closing tag. CSS is a language that defines the style for each class of element. For example: all h1 elements w/ class = "article" should be red in color.

Boxes, Grids & Rules

Everything on the web is boxes! Boxes are easy to arrange and rearrange on a background page grid.

Boxify a Design

Start with one big box around the material. Decompose the layout naturally with smaller and smaller subboxes. This will make converting to HTML simpler. BIG to small.
